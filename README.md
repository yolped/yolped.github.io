# Personal Portfolio

---

This repository contains my personal portfolio website built using only HTML, CSS and vanilla JavaScript. The portfolio is lightweight, responsive, and hosted using GitHub Pages.

**Live URL:** `https://yolped.github.io` <!-- replace with your live URL -->

---

## Features

- Responsive design
- Intelligent theme detection — follows system/browser preference instantly with no flash or persistence
- Mobile-first
- Fast loading
- Resume download
- No frameworks
- Accessible
- GitHub Pages deployment

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

---

## Project Structure

```text
.
├── index.html
```

---

## Running Locally

1. Clone the repository.
2. Open `index.html` in any modern browser.

No build tools or installation required.

---

## Customization

The portfolio can easily be customized by updating:

- Personal information
- Resume
- GitHub
- LinkedIn
- Projects
- Experience

---

## Deployment

The website is deployed using GitHub Pages and automatically updates whenever changes are pushed to the `main` branch.
